# 🏙️ CityLens

> Find your next move. CityLens analyzes foot traffic, market saturation, and customer demographics — in 30 seconds.

Built for the **Grab Hackathon 2026** — Smarter Places track.

## 🎯 The Problem

A business owner thinks: *"I want to expand but I have no idea which neighborhood is right for me."* Currently they'd spend days googling, visiting spots, guessing. CityLens answers this in 30 seconds with a beautiful visual interface.

## 🚀 Built With

- React 18 + Vite
- Tailwind CSS
- Lucide Icons
- GrabMaps APIs (production)

## 🛠️ Run Locally

```bash
npm install
npm run dev
```
